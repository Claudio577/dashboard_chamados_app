# Dashboard de Chamados

Sistema em Streamlit para gerar dashboard mensal de chamados.

## Funcionalidades

- Upload de planilha Excel ou CSV
- Total de chamados
- Total de empresas com chamados
- SLA tratado
- Dentro do SLA
- Fora do SLA
- Chamados tratados em até 72 horas
- Chamados acima de 72 horas ou em aberto
- Empresa com mais chamados
- Qualificação mais solicitada
- Gráfico de pizza de 72 horas
- Tabelas de conferência
- Download do dashboard em Excel
